# Module 3 Lecture 1: Memory — Comprehensive Study Notes

## 1. What is Memory? [EXAM]

**Memory** is the series of processes whereby the **nervous system** acquires information from new experiences, retains this information over time, and eventually uses it to **guide behavior** and **plan future actions**.

- This definition comes directly from the textbook (Principles of Cognitive Neuroscience, Ch. 8)
- [PROF EMPHASIS] The professor stressed that memory encompasses virtually **any type of information** — facts, events, skills, riding a bike, etc.
- The textbook adds that **learning** is a closely related term, used as a synonym for encoding, and can also describe gradual changes in behavior as a function of training

---

## 2. Three Phases of Memory [EXAM]

Memory is broadly defined in **three phases**:

### 2.1 Encoding
- The process of **acquiring new information** that leaves a **memory trace**
- [EXAM] **Memory traces** = alterations (changes) in the **strength and/or number of synaptic connections** between neurons
- Example: Studying a list of words in the lab, or attending a lecture and absorbing new material
- The professor likened encoding to "being exposed to new information of some sort that then leaves some kind of memory trace"

### 2.2 Storage
- The **retention** of memory traces over time
- Long-term retention requires **consolidation** processes (both cell-level and system-level)
- [EXAM] Key difficulty: it is hard to prove memories are truly "lost" during storage — a memory that cannot be retrieved may still exist but simply be **inaccessible** (available but not accessible distinction)
- Animal models show that synaptic changes can reverse over time in parallel with behavioral forgetting, suggesting some memories genuinely are lost

### 2.3 Retrieval
- **Accessing** stored memory traces at a later point in time
- May lead to a change in behavior and is sometimes associated with the **conscious experience of remembering**
- Example: Being examined on lecture material weeks later = retrieval
- [PROF EMPHASIS] The professor emphasized that retrieval is not just exams — daily life involves constant retrieval of memories in less structured ways

## 3. Types of Memory [EXAM]

### 3.1 Declarative vs. Nondeclarative Memory

The fundamental division:

| **Declarative (Explicit)** | **Nondeclarative (Implicit)** |
|---|---|
| Conscious recollection | No conscious recollection required |
| Episodic + Semantic memory | Priming, skill learning, conditioning |
| Depends on **medial temporal lobe** | Spared in MTL damage (cf. H.M.) |

### 3.2 Episodic Memory [EXAM]
- Memory for **specific events** — includes what happened, where, when, and how
- Heavily dependent on the **hippocampus** and broader **medial temporal lobe**
- Studies focus on both **encoding and retrieval** phases
- Research emphasis on **memory reactivation**: whether cortical regions engaged during encoding are recruited again during retrieval

### 3.3 Semantic Memory
- Memory for **general knowledge and facts** (e.g., "Paris is the capital of France")
- Studies typically focus on knowledge acquired **before** the experiment
- Research emphasis on **memory organization** in the cortex
- Two main theories of cortical organization:
  - **Sensory/functional theory** (Warrington, Shallice): knowledge organized by sensory (form, color) and functional (motions, actions) properties — instance of **embodied cognition**
  - **Domain-specific theory** (Caramazza): concepts organized by semantic categories with evolutionary significance (animals, fruits/vegetables, conspecifics, nonliving things)

### 3.4 Autobiographical Memory
- [EXAM] Overlap between **episodic** and **semantic** memory
- Memories of your **own life** — helps with storytelling and creates a **sense of self**
- Depends on the **medial temporal lobe**
- [PROF EMPHASIS] Example from lecture: You might remember learning "Paris is the capital of France" as an episode (episodic), but over time the episodic context fades and it becomes pure semantic knowledge

### 3.5 Familiarity vs. Recollection
- **Familiarity**: feeling you have encountered something before without remembering specifics (e.g., recognizing a face but not recalling the person's name)
- **Recollection**: full conscious retrieval of the event/context
- [EXAM] Classification of familiarity as episodic memory is **debated** — its neural mechanisms relate more to semantic memory and priming

## 4. The Neurological Model of Memory [EXAM]

### 4.1 The Hippocampus as an Index [EXAM — HIGH YIELD]

The textbook presents a **simple but critical model** (Figure 9.2):

**During Encoding:**
- Different aspects of an event are stored in the **cortical regions** that processed them:
  - Visual aspects → **visual cortex**
  - Auditory aspects → **auditory cortex**
  - Motor aspects → **motor cortex**
- [EXAM] The **hippocampus** stores a **summary representation** (an **index**) with **pointers** to where distributed cortical traces are located
- [PROF EMPHASIS] The professor used the analogy: "Like the index of a book — the hippocampus doesn't store the memory itself, it stores pointers to where the memory is in the brain"
- Two types of representations are created: **distributed cortical traces** + **hippocampal indices**

**During Storage:**
- Some memory traces are **strengthened** = **consolidation**
- Others are presumably lost over time
- The hippocampus interacts with **PFC** and **PPC** during this process

**During Retrieval:**
- A **retrieval cue** (external or internal) accesses the hippocampal index
- This leads to simultaneous access of distributed cortical traces → experience of remembering
- Retrieval can be **voluntary** (intentional effort) or **involuntary** (spontaneous)
- [EXAM] For **recent memories**: retrieval is mediated through the hippocampus
- [EXAM] For **remote memories**: direct cortical associations may allow retrieval **bypassing the hippocampus**

### 4.2 Why MTL Damage Is So Devastating [EXAM]

The model explains three features of memory loss after brain damage:

1. **Small cortical lesions** → mild or no memory deficits (traces are distributed; one lesion unlikely to destroy associated traces — relates to **graceful degradation**)
2. **Bilateral MTL lesions** → severe **amnesia** (the hippocampal index is concentrated in a small region; damage blocks access to vast numbers of cortical traces)
3. This is the key insight from the **case of H.M.** (Chapter 8): MTL damage impairs **declarative memory** but spares **nondeclarative memory** (priming, skill learning)

## 5. The Hippocampus and Medial Temporal Lobe [EXAM]

### 5.1 Hippocampal Subregions
- The hippocampus is subdivided into different subregions with specific sub-functions
- Main regions: **CA1, CA3, dentate gyrus, subiculum**
- [PROF EMPHASIS] "Don't memorize each arrow — remember that these regions are **heavily intertwined** with each other and also with **other MTL regions**"
- They interact to enable you to **memorize things in a coherent fashion**

### 5.2 Other MTL Regions
- **Amygdala**: involved in **emotional memory** — interacts heavily with the hippocampus
  - Emotion is often related to memory because retrieved events are frequently not neutral
  - Both amygdala and hippocampus are involved when emotion + memory intersect
- **Parahippocampal cortex**: attached to the hippocampus within the MTL; contributes to contextual processing

### 5.3 Theories of Hippocampal Function [EXAM]
Three major theories, all supported by evidence:

1. **Cognitive map theory**: hippocampus supports spatial memory/navigation
2. **Relational memory theory**: hippocampus binds relationships between items/elements
3. **Episodic memory theory**: hippocampus is critical for episodic encoding and retrieval

- Possible integration: **spatial** functions more pronounced in **right hippocampus**, general **relational** functions in **left hippocampus**
- Both relational and episodic theories are consistent with linking the hippocampus to **recollection**

## 6. Prefrontal Cortex (PFC) in Memory [EXAM]

### 6.1 PFC Functions in Memory
- [EXAM] PFC is known for **many functions** that relate to memory directly or indirectly
- **During encoding**: PFC helps **link incoming stimuli with preexisting knowledge** and **organize them into meaningful units** for easier remembering
- **During retrieval**: PFC helps **guide memory search**, **evaluate recovered information**, and supports **decisions about validity**
- PFC is less critical during storage but may play a role in consolidation

### 6.2 The Subsequent Memory Paradigm [EXAM — HIGH YIELD]

A specific, controlled paradigm for studying **episodic memory encoding**:

**Four Steps:**
1. Participants study a series of items (e.g., word list) while brain activity is recorded
2. Later memory test — participants remember some, forget others
3. Encoding trials sorted into **subsequently remembered** vs. **subsequently forgotten**
4. Compare brain activity between these two categories

- [EXAM] Greater activity for remembered > forgotten = **Subsequent Memory Effects (SMEs)**
- First used in ERP studies in early 1980s by **Kenneth Paller** and others
- SMEs tend to appear over frontal and parietal electrodes starting ~300 ms
- [EXAM] Key finding: **Inferior frontal gyrus** (a PFC subregion) is specifically involved
  - Also: **medial temporal lobe** and **dorsal parietal lobe**

### 6.3 PFC During Retrieval
- [PROF EMPHASIS] Unlike encoding (where inferior frontal gyrus is specifically implicated), during retrieval **multiple PFC subregions** are involved — "it's not so specific as the encoding example"
- The **rostral prefrontal cortex** is also involved in retrieval

## 7. Posterior Parietal Cortex (PPC) in Memory [EXAM]

### 7.1 Dorsal vs. Ventral PPC

| **Dorsal PPC** | **Ventral PPC** |
|---|---|
| **Memory search** during retrieval | **Salience** of memories |
| General **monitoring** / allocating **attentional resources** | Bottom-up attention to retrieval cues |
| Top-down attentional control | During encoding: associated with **subsequent forgetting** |

- PPC supports **attentional control processes** critical for:
  - **Selecting information** during encoding
  - **Guiding memory search** during retrieval
- [EXAM] The encoding data (subsequent memory paradigm) also shows **parietal cortex involvement** — not just PFC

## 8. Systems Consolidation [EXAM — HIGH YIELD]

### 8.1 Standard Consolidation Theory [EXAM]

The most prominent theory of how memories become independent of the hippocampus:

**Timeline:**
1. **Recent memories**: Hippocampus stores the index; interacts with PFC and PPC
   - The hippocampus **binds** different aspects of an event together
   - PFC and PPC store different aspects of the event
2. **Over time**: Each time a memory is retrieved (via a retrieval cue), the hippocampal trace is reactivated
3. **Gradually**: Direct connections form among cortical traces (PPC, PFC)
4. **Remote memories**: Cortical regions can reconstruct the memory **without needing the hippocampus**
   - [EXAM] This explains why **remote memories are often spared** by hippocampal damage

- [PROF EMPHASIS] The professor illustrated: "If I ask about this weekend → hippocampus lights up strongly. But a weekend from months ago → cortical regions (PPC, PFC) might handle it on their own"

### 8.2 Multiple Trace Theory [EXAM]

**Directly opposes standard consolidation theory:**

- [EXAM] Core claim: **Episodic memories are ALWAYS dependent on the hippocampus** — no matter how old or remote
- Each time you encode or retrieve an event (even the same event), you create a **new trace** of that event
- The new trace overlaps largely with previous ones but may be somewhat different
- Because you always store a new trace, you **always need the hippocampus** to store it and the relations between its parts
- [PROF EMPHASIS] "The funny thing is you might think this is easily testable" — but it's very difficult to test empirically

### 8.3 Consolidation Is Promoted By:
- **Intentional practice** (repeated study/retrieval)
- **Slow-wave sleep** — memory reactivation during sleep
- **Rest** — consolidation can occur during quiet waking periods

## 9. Types of Memory Retrieval Tasks [EXAM]

Three common retrieval task types mentioned in the lecture:

1. **Free recall**: Retrieve as many items as possible from a studied list
2. **Recognition**: Decide whether an item was in the studied list or not
3. **Source memory test**: Identify which list/context an item belonged to

- [PROF EMPHASIS] "You can make a very long list of retrieval tasks" — these are just the most common
- Regardless of task type, **PFC regions** are heavily involved (not just MTL/hippocampus)

## 10. Working Memory [EXAM]

### 10.1 The Baddeley Model [EXAM]

Proposed by psychologist **Alan Baddeley** in the 1970s:

**Three capacity-limited memory buffers:**
1. **Phonological loop** — holds **sound-based** representations
2. **Visuospatial sketchpad** — holds **visual object and spatial** representations
3. **Episodic buffer** — holds **integrated, multimodal** representations

**Control system:**
- **Central executive** — allocates processing resources to buffers and performs manipulations

**Each buffer has two components:**
- A **store** that holds information briefly
- A **rehearsal mechanism** that reactivates information before it dissipates

**Connections to long-term memory:**
- Phonological loop ↔ Language
- Visuospatial sketchpad ↔ Visual semantics
- Episodic buffer ↔ Episodic memory

## 11. Key Brain Regions Summary Table [EXAM]

| **Brain Region** | **Role in Memory** |
|---|---|
| **Hippocampus** | Stores index/pointers to cortical traces; binds aspects of events; critical for encoding & retrieval of recent memories |
| **Medial Temporal Lobe** (broader) | Stores pointers; damage causes severe amnesia |
| **Amygdala** | Emotional memory; interacts with hippocampus |
| **Parahippocampal cortex** | Context processing within MTL |
| **Prefrontal cortex (PFC)** | Encoding: links stimuli to prior knowledge; Retrieval: guides search, evaluates information |
| **Inferior frontal gyrus** | Specifically involved in encoding (subsequent memory effects) |
| **Posterior parietal cortex (PPC)** | Attentional control; dorsal = memory search/monitoring; ventral = salience/bottom-up attention |
| **Visual/Auditory/Motor cortex** | Store modality-specific aspects of memory traces |

## 13. Key Takeaways

1. **Memory has three phases** — encoding (acquiring), storage (retaining), and retrieval (accessing) — each involving distinct brain mechanisms and regions [EXAM]

2. **The hippocampus functions as an index** — it stores pointers to distributed cortical traces, not the memories themselves. This explains why small MTL lesions cause devastating amnesia while cortical lesions do not [EXAM]

3. **Standard consolidation theory vs. Multiple trace theory** is a central debate: standard theory says remote memories become hippocampus-independent; multiple trace theory says the hippocampus is ALWAYS needed [EXAM]

4. **The subsequent memory paradigm** is the key method for studying encoding — comparing brain activity for subsequently remembered vs. forgotten items reveals the **inferior frontal gyrus** and MTL as critical for successful encoding [EXAM]

5. **PFC has distinct roles in encoding vs. retrieval** — during encoding, the inferior frontal gyrus is specifically involved; during retrieval, multiple PFC subregions participate [EXAM]

6. **PPC contributes via attentional mechanisms** — dorsal PPC supports memory search and monitoring; ventral PPC responds to salience and bottom-up attention [EXAM]

7. **Episodic, semantic, and autobiographical memory** are distinct but overlapping systems — autobiographical memory bridges episodic and semantic, and the transition from episodic to semantic knowledge occurs naturally over time [EXAM]

8. **Baddeley's working memory model** (phonological loop, visuospatial sketchpad, episodic buffer, central executive) explains how information is temporarily maintained and manipulated [EXAM]
